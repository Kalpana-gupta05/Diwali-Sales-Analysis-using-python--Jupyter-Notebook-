# Diwali-Sales-Analysis-using-python--Jupyter-Notebook-
"End-to-End Data Analysis project on Diwali Sales dataset including data cleaning, EDA, and visualization using Python."
## Project Overview
This project focuses on analyzing Diwali Sales data to uncover customer purchasing behavior, identity key trends, and generate insights that can help improve sales stragies and customer experience.
The analysis is performed using Python and popular data analysis libraries.

## Objective 
- Understand customer demograohics
- Identify top-performing product categories
- Analyze purchasing patterns by gender, age, and location
- Generate actionable business insights

## Tools and Technologies Used
- Python
- Pandas
- Numpy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The dataset contains informatin about:
- Customer demographics (Gender, Age, Marital Status)
- Product categories
- Purchase amount
- City and state date

## Key Analysis Performed
- Data cleaning and preprocessing
- Handling missing values
- Exploratory Data Analysis (EDA)
- Visualization of trends and patterns

## Key Insights
- Married women (age group 26- 35) are the highest buyers
- Most sales come from specific states (e.g.,Uttar Pradesh, Maharashtra)
- Top categories include Food, Clothing, and Electronics
- Customers from IT, Healthcare, and Aviation sectors spend more

## Why This Project Matters
This project demonstrates:
- Data cleaning and preprocessing skills
- Exploratory Data Analysis (EDA)
- Data visualization technoques
- Business insights generation
